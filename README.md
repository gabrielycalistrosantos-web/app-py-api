# app-py-api
API simples desenvolvida com Python + Flask que permite realizar operações de CRUD em uma coleção de livros

A API possibilita:
- Consultar todos os livros;
- Consultar livros por ID;
- Adicionar novos livros;
- Alterar informações de livros por ID;
- Deletar livros por ID;

Tecnologias utilizadas: 
- Python 3;
- Flask;
- Json;
- Ambiente Virtual (venv);

Conceitos aplicados:
- Criação de API REST;
- Manipulação de JSON;
- Métodos HTTP (GET, POST, PUT, DELETE);
- Uso de Flask;
- Estrutura básica de Backend

Melhorias futuras:
- Inclusão de banco de dados;
- Validação de dados;
- Tratamento de erros;
